阅读时间生成

vuepress-plugin-reading-time1

预计阅读时间与字数统计生成

这个插件将会向 page 对象注入预计阅读时间与字数统计。

会自动向 page 对象注入 readingTime:

interface ReadingTime {
  /** 分钟数 */
  minutes: number;
  /** 字数 */
  words: number;
}
