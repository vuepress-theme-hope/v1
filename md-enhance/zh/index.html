vuepress-plugin-md-enhance

为 VuePress 提供更多 Markdown 增强功能

快速上手 💡配置 🛠

自定义容器支持

用样式装饰 Markdown 内容

选项卡

使用选项卡对相似内容进行分组

代码组

使用选项卡对相似代码进行分组

自定义对齐支持

让你在 Markdown 中对段落进行对齐

上下角标支持

你的 Markdown 现在将支持上下角标

脚注

你的 Markdown 现在将支持脚注

标记支持

让你在 Markdown 中对词句进行标记

任务列表

在 Markdown 中使用任务列表

图片语法

使用改进的语法指定图片大小与颜色模式

图表支持

在 Markdown 中展示图表

流程图支持

可以在 Markdown 中直接写出流程图

Mermaid 支持

可以在 Markdown 中添加 Mermaid 图例

Tex 支持

Markdown 现在也可以支持 Tex 语法以显示公式

导入文件支持

将你的文档分段，并在 Markdown 中导入

代码案例支持

你可以很方便的插入代码案例

幻灯片支持

你可以直接在 Markdown 中插入幻灯片
