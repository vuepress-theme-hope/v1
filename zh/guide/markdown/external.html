# Markmap

Markmap 是一个将 Markdown 转换为思维导图的工具。它相对于主题内置的 flowchart 支持更多的格式与内容。

  1. 使用 Markmap (opens new window) 生成思维导图 HTML 文件
  2. 将 HTML 文件放在 .vuepress/public/ 下
  3. 通过 <iframe> 插入到 Markdown

输入:

<iframe
  :src="$withBase('/markmap/demo-zh.html')"
  width="100%"
  height="400"
  frameborder="0"
  scrolling="No"
  leftmargin="0"
  topmargin="0"
/>
